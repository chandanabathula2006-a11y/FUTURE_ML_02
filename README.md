# Support Ticket Classification System

## Overview

This project is a Machine Learning and Natural Language Processing (NLP) based system that automatically classifies customer support tickets and assigns priority levels. It helps support teams manage tickets more efficiently and respond to customer issues faster.

## Project Objective

To build a ticket classification system that can analyze support requests, predict their category, and assign an appropriate priority level.

## Key Features

* Text Cleaning and Preprocessing
* Tokenization of Ticket Text
* TF-IDF Vectorization
* Ticket Category Classification
* Priority Tagging (High, Medium, Low)
* Model Performance Evaluation

## Technologies Used

* Python
* NLTK
* Scikit-learn
* Pandas
* Jupyter Notebook

## Workflow

Customer Ticket → Text Preprocessing → TF-IDF Vectorization → Model Training → Ticket Classification → Priority Assignment

## Skills Gained

* Natural Language Processing (NLP)
* Text Preprocessing
* Machine Learning Classification
* Feature Extraction
* Model Evaluation

## Outcome

Developed a ticket classification system that helps automate support ticket management and improve response efficiency.
